🎣 Phishing Awareness Demo - Cyber Drill

🔍 What is Phishing?

Phishing is a deceptive cyberattack where attackers mimic trusted sources to steal sensitive data like passwords or bank details.

🎯 Demo Objective

This simulation educates users on how phishing sites work — it’s 100% safe and does not store real data.

📝 Instructions

  • Click on Try Now
  • You will see a fake login screen
  • Enter any fake credentials
  • You’ll then be warned it was a phishing demo

🛡️ Awareness Tips

  • Check the URL carefully before logging in
  • Don’t click unknown or urgent links
  • Use two-factor authentication
  • Never reuse passwords
